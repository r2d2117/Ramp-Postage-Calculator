How to use this API

API calls to this service are made as follows:

  • the URL talkes the form /units/length/width/weight
  • units is either 'metric' or 'imperial'
  • Length, width, and weight are all non-zero positive numbers

Here are some example API calls:

  • http://localhost:4000/metric/370/5/110
  • http://localhost:4000/metric/300/91/49
  • http://localhost:4000/metric/150/91/49
